Powerful Capabilities

More than just a chatbot. VocalEye interacts with your computer.

Content Gen

Create essays, reports, or summaries. Automatically saves as .PDF or .DOCX.

OS Control

"Open Chrome", "Launch Notepad". VocalEye intelligently scans your installed applications and executes commands.

Smart Emailing

Dictate the recipient and topic. VocalEye drafts a professional email using LLM intelligence and sends it via SMTP.

First Run Configuration

1

Get your Gemini API Key

VocalEye requires Google's Generative AI to think. It's free to get a key.

Get Gemini Key →
2

Create an App Password

To send emails, you need a Google App Password (not your regular login password).

Create App Password →
3

Launch & Enter Credentials

Run the .exe file. On the first launch, a configuration window will appear. Paste your keys there. Your credentials are stored securely in your system's Keyring (Credential Manager).
