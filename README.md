## ğŸš€ Getting Started

### Prerequisites
Before running VocalEye, you will need:
1.  **Gemini API Key**: Get it for free at [ai.google.dev](https://ai.google.dev/).
2.  **Google App Password**: Required for email functionality. Generate one at [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords).

### Installation & Run
1.  Download the latest release for your OS (Windows or Linux).
2.  Run the executable (`.exe` on Windows).
3.  On the first launch, a configuration window will appear.
4.  Enter your **Gemini API Key** and **Google App Password**.
5.  Credentials are securely stored in your system's Keyring.
